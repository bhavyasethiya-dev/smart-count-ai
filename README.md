# 🤖 SmartCount AI

### Intelligent Item Counting System using AI & Computer Vision

SmartCount AI is a full-stack web application that automatically detects and counts objects in images or video streams using AI-powered computer vision models. It helps businesses and individuals efficiently track items without manual effort.

---

## 🚀 Features

* 🔍 Automatic object detection and counting
* 📸 Supports image upload (extendable to video/live feed)
* ⚡ Fast processing with real-time results
* 🌐 Full-stack web app (frontend + backend)
* 📊 Displays counted results clearly on UI
* 🔐 Scalable backend with API support

---

## 🔄 How It Works

1. User uploads an image
2. Image is sent to backend API
3. AI model processes the image
4. Objects are detected and counted
5. Result is returned and displayed on frontend
